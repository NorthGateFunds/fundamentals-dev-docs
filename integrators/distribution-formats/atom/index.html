Implementation notes

Idempotency
Use the Atom <id> as the primary key. Entries may be re-emitted when metadata changes; treat ingestion as upserts.
Content
Atom entries can include XHTML-safe content. If your client can’t render XHTML, fall back to <summary> and the canonical URL.
Canonical URL
Always follow the canonical link rel="alternate" for the authoritative public release.
