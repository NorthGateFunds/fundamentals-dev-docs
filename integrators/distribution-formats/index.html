Distribution Formats

Fundamentals Wire distributes press releases through multiple machine-readable formats. Each format serves a distinct integration purpose while referring to the same canonical, immutable release.

NewsML (Primary)

NewsML is the canonical, full-fidelity disclosure format used by Fundamentals Wire. Each published press release is delivered as a complete NewsML 1.2 document via HTTPS push to registered integrator endpoints.

View NewsML HTTPS Push integration guide →

RSS

RSS feeds are intended for broad compatibility and low-friction polling-based ingestion. Each item represents a published press release and links to the canonical public URL.

Atom

Atom feeds provide stronger semantics around identifiers and timestamps. Atom is recommended when strict ordering and update detection matter.

Example outputs

Format-specific pages include real example payloads generated by the Fundamentals Wire publication pipeline so integrators can validate ingestion behavior against production output.

Current state: published releases are immutable. Corrections and superseding releases are not implemented yet.
